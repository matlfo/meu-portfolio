Portfólio pessoal desenvolvido com React, Framer Motion e Vite para apresentar minhas habilidades, projetos e experiências na área de TI e Desenvolvimento.

## 🚀 Demo

🔗 **[Acessar o Portfólio Online](https://matlfo.github.io/meu-portfolio/)**

## 🎯 Funcionalidades

- ✨ Animações fluidas e interativas com Framer Motion
- 📱 Design 100% responsivo (mobile-first)
- ⚡ Carregamento otimizado e rápido com Vite
- 🎨 Tema escuro moderno com interface limpa
- 📧 Integração de contato direto

## 🛠️ Tecnologias Utilizadas

- **React 18** - Framework JavaScript
- **Framer Motion** - Biblioteca de animações
- **Lucide React** - Ícones modernos
- **Vite** - Build tool e dev server
